Ex01 : Processus d'affichage d'une page statique.

Lorsque django reçoit la requete pour une page web statique, le fichier urls.py du projet puis de l'app determine quelle est la fonction qui correspond a cette requete dans le fichier views de l'app. Cette fonction va renvoyer une réponse HTTP apres avoir processé le template concerné grace au moteur de template afin d'obtenir le code HTML.
