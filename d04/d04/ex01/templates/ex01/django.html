Ex01 : Django, framework web.

Django qu'est-ce que c'est ?

Django est un framework web open source en python pour rapidement créer des sites web. Son slogan est: "Le framework web pour perfectionnistes sous pression". Il s'inspire du principe MVC arrangé en MTV: Models, Templates (gabarits), Views.

Sont inclus dans ce framework également:

  • API d'accès aux données
  • Un serveur web léger pour le développement et les tests
  • Intégration d'un système de traitement de formulaire
  • Et bien d'autres plugin

Historique

Conçu en 2003 pour un journal local du Kansas, il a depuis eu de nombreux contributeurs. Depuis 2008, la Django Software Foundation s'occupe de la promotion du cadre. De nombreux sites utilisent Django dont Instagram, Mozilla, The Washington Times, Bitbucket...
